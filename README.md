> ### asciify
> convert gif files to ascii (soon jpg / png)
---
> ### setup
> install [deno](https://deno.land)<br>
> import `https://github.com/215B5D/asciify/raw/master/mod.ts`
---
> ### usage
> check `example.ts` for an example<br>
> `deno run -A example.ts`
